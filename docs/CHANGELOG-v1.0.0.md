## [1.0.0] - 2026-01-29 🎉

First release!

### ✨ Features

- 🃏 Visual 13x13 hand matrix with action-based coloring
- 📁 File browser with fast search/filter
- 🎨 Color-coded actions and legend
- 📝 Details panel with strategic notes
- 📂 Directory support (auto-loads all `.yaml` files)

### 📦 Builds

- 🐧 Linux (amd64)
- 🍎 macOS (Intel & Apple Silicon)
- 🪟 Windows (amd64)
